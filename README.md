# Polling Station Finder

A web app that lets users search for their polling station by typing an address. Built with FastHTML, HTMX, and Leaflet.

- **Search** — start typing an address; results appear as you type
- **Map** — shows all polling district boundaries; selecting an address highlights your district and drops a pin on your polling station
- **Directions** — each result links to Google Maps for walking/driving directions

## Documentation

Full design journey, data pipeline, and route documentation is in [`polling_app_clean.ipynb`](polling_app_clean.ipynb).

## Working version

The running app code is in [`polling_app4.ipynb`](polling_app4.ipynb).

## Running it

Open either notebook in Solveit, restart the kernel, and run all cells. Visit `/search_page` on the app's public URL.
